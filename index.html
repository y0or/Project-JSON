üíª Practical Examples

1. Creating and displaying JSON in Node.js

// Create an object with data
const student = {
  name: "Lucas",
  age: 20,
  course: "IT"
};

console.log("Object:");
console.log(student);

console.log("Converted to JSON:");
console.log(JSON.stringify(student, null, 2));

2. Reading JSON data (e.g. from an API)

const jsonReceived = '{"name":"Ana","age":22}';
const obj = JSON.parse(jsonReceived);
console.log(obj.name); // Ana

3. Interactive examples in Node.js with JSON

Example 3.1: Creating an object and converting to JSON

const user = {
  name: "Mariana",
  age: 27,
  active: true,
  interests: ["music", "sports", "reading"]
};

const jsonText = JSON.stringify(user, null, 2);
console.log("JavaScript Object:");
console.log(user);

console.log("\nConverted to JSON:");
console.log(jsonText);

Explanation: We use JSON.stringify to convert the JS object to formatted JSON text.

Example 3.2: Converting JSON to object and accessing data

const jsonData = '{"product":"Book","price":29.9,"available":true}';
const product = JSON.parse(jsonData);

console.log("Product name:", product.product);
console.log("Price:", product.price);
console.log("Available?", product.available);

Explanation: Using JSON.parse we convert JSON text into a JS object to access the data.

Example 3.3: Simulating a mini JSON database

const users = [
  { id: 1, name: "Lucas", active: true },
  { id: 2, name: "Ana", active: false },
  { id: 3, name: "Pedro", active: true }
];

const jsonUsers = JSON.stringify(users, null, 2);
console.log("List of users in JSON format:");
console.log(jsonUsers);

const usersObj = JSON.parse(jsonUsers);
const activeUsers = usersObj.filter(u => u.active);

console.log("\nActive users:");
console.table(activeUsers);

Explanation: Shows how JSON can represent complex lists and uses console.table to display data.

Example 4: Nested JSON with address

const person = {
  name: "Pedro",
  age: 30,
  address: {
    street: "Av. Brasil",
    number: 100,
    city: "Rio de Janeiro",
    state: "RJ",
    zipcode: "20000-000"
  }
};

console.log(JSON.stringify(person, null, 2));

Explanation: This example shows how JSON can represent objects inside objects (nested data).

Example 5: Filtering a list of JSON objects

const products = [
  { id: 1, name: "Pen", price: 1.2, inStock: true },
  { id: 2, name: "Notebook", price: 4.5, inStock: false },
  { id: 3, name: "Pencil", price: 0.8, inStock: true }
];

const availableProducts = products.filter(p => p.inStock);

console.log(JSON.stringify(availableProducts, null, 2));

Explanation: Filters a JSON array to get only products that are in stock.
