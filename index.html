Example 3.1: Creating an object and converting to JSON

  name: "Mariana",

Example 3.2: Converting JSON to object and accessing data

const jsonData = '{"product":"Book","price":29.9,"available":true}';
const product = JSON.parse(jsonData);

console.log("Product name:", product.product);
console.log("Price:", product.price);
console.log("Available?", product.available);

Explanation: Using JSON.parse we convert JSON text into a JS object to access the data.

Example 3.3: Simulating a mini JSON database

const users = [
  { id: 1, name: "Lucas", active: true },
  { id: 2, name: "Ana", active: false },
  { id: 3, name: "Pedro", active: true }
];

const jsonUsers = JSON.stringify(users, null, 2);
console.log("List of users in JSON format:");
console.log(jsonUsers);

const usersObj = JSON.parse(jsonUsers);
const activeUsers = usersObj.filter(u => u.active);

console.log("\nActive users:");
console.table(activeUsers);

Explanation: Shows how JSON can represent complex lists and uses console.table to display data.

Example 5: Filtering a list of JSON objects

üåê Example usage with API using fetch

The fetch() function is used to get data from an API (a web server).

When the data arrives, we convert it from JSON text to a usable object using .json().

fetch("https://jsonplaceholder.typicode.com/users")
  .then(res => res.json()) // Converts the response text into a JavaScript object
  .then(data => {
    console.log("Users from API:");
    console.table(data); // Shows the user list in table format in the console
  });

Example:

  • fetch(url): Sends a request to the URL.
  • .then(res => res.json()): Converts the response (which is in JSON format) into an object.
  • .then(data => ...): This is where you can use the data received.

‚úÖ This only works in browsers, because it needs internet access to contact the API.
